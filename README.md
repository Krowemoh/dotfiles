# dotfiles

This is my dotfiles.

```
git clone https://github.com/Krowemoh/dotfiles.git 
```

You need to first install the base system:

```
base
linux
linux-firmware
efibootmgr
grub
vim
openssh
sudo
networkmanager
base-devel
linux-headers
git
rust
```

Then install paru and pacmanfile:

```
paru
pacmanfile
```

For use with stow:

```
cd ~/dotfiles
stow fish
stow sway
stow foot
stow vim
```
